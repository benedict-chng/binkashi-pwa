  <action>
Add image thumbnail display to BinCard component:

1. Create Blob URL from image Blob:
   - Add state for image URL: `const [imageUrl, setImageUrl] = useState<string | null>(null);`
   - In useEffect, create Blob URL: `if (bin.image) setImageUrl(URL.createObjectURL(bin.image))`
   - Cleanup in useEffect return: `if (imageUrl) URL.revokeObjectURL(imageUrl)`
   - This prevents memory leaks (per STACK.md research requirement)

2. Add thumbnail UI:
   - Display image above bin name if imageUrl exists
   - Use Tailwind classes for responsive thumbnail: `w-full h-48 object-cover rounded-t-lg bg-gray-200`
   - Add cursor pointer to indicate clickable: `cursor-pointer hover:opacity-90 transition-opacity`
   - Handle missing image state: show placeholder or no image element

3. Add onImageClick prop to BinCardProps:
   - Accept callback: `onImageClick?: (imageUrl: string) => void;`
   - Call callback on image click: `onClick={() => onImageClick?.(imageUrl)}`
   - This allows parent (BinList) to control modal state

4. Styling adjustments:
   - Move image to top of card for better visual hierarchy
   - Add rounded-t-lg to image, rounded-b-lg to card body for seamless appearance
   - Maintain existing spacing and typography for text content

Critical: Must revoke Blob URLs to prevent memory leaks (per research findings on image Blob memory leaks).
