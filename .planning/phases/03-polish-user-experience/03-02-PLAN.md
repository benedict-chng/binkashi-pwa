Create image compression utility using HTML Canvas API:

1. **compressImage(file, options)**:
   - Accepts File or Blob object
   - Options: maxWidth (default 1200), quality (default 0.8), maxSizeKB (default 500)
   - Returns Promise<Blob> with compressed image

2. **Compression algorithm**:
   - Load image into Canvas API
   - Resize to maxWidth if image is larger
   - Compress with toBlob() and quality parameter
   - If result exceeds maxSizeKB, recursively reduce quality until under limit or minimum quality (0.5)
   - Support JPEG and PNG formats

3. **Error handling**:
   - Handle invalid file types (not images)
   - Handle corrupted images
   - Return original file if compression fails

4. **Configuration constants**:
   - MAX_IMAGE_WIDTH: 1200 (good balance of quality and file size)
   - MAX_IMAGE_SIZE_KB: 500 (500KB per image = ~50MB for 100 images)
   - DEFAULT_QUALITY: 0.8 (80% quality)

Use TypeScript for type safety. Use native Canvas API (no external libraries like browser-image-compression).
