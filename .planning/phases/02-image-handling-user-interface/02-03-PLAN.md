<task type="auto">
  <name>Task 1: Create Toast component and context</name>
  <files>src/components/Toast.tsx</files>
  <action>
Create Toast notification system with React Context for app-wide feedback:

1. Create Toast component:
   - Toast interface: `type Toast = { id: string; message: string; type: 'success' | 'error' | 'info'; duration?: number }`
   - Display style: Fixed position (top-right or bottom-center), colored background by type
   - Success: green background, Error: red background, Info: blue background
   - Auto-dismiss after duration (default 3000ms)
   - Animation: slide in/fade in on mount, fade out on unmount

2. Create ToastContext:
   - Context type: `interface ToastContext { showToast: (message: string, type: 'success' | 'error' | 'info', duration?: number) => void }`
   - Provider component: Manages toasts array with add/remove functions
   - useToast hook: Exposes showToast function for components

3. Implementation details:
   - Use setTimeout for auto-dismiss, clear timeout on unmount
   - Use unique IDs (timestamp or random) for toast identification
   - Limit max toasts displayed (e.g., 3) to prevent overwhelming UI
   - Tailwind classes: `fixed top-4 right-4 z-50 space-y-2` for positioning

4. TypeScript safety:
   - Export ToastContext and useToast hook
   - Throw error if useToast called outside ToastProvider (standard pattern)

Toast system follows React Context pattern for global state without external libraries.
  </action>
